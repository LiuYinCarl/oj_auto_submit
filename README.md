# oj_auto_submit

# OJ自动提交工具

--------------------------------

##### POJ自动提交工具使用指南

- 首先用pip安装好所需要的模块 pip3 install requests lxml prettytable
- 根据readme文件中的说明对poj_conf.json进行更改

- 使用命令行执行程序，当修改好配置文件后直接在命令行中执行程序，Pytho程序如何执行不做说明
